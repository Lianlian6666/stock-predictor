# 📈 智能股票预测助手

## 项目概述

一个基于多维度量化分析的股票预测Web应用，支持手机端使用，可安装为PWA（渐进式Web应用）。

## 核心功能

### 🔮 多维度预测模型

采用 **四大维度加权评分体系**：

| 维度 | 权重 | 包含指标 |
|------|------|----------|
| **技术面** | 40% | 均线系统(MA5/10/20/60)、MACD、RSI、KDJ、布林带(BOLL)、成交量趋势、动量分析 |
| **基本面** | 25% | 市盈率(PE)、市净率(PB)、ROE、营收增长率、净利润增长率、毛利率、EPS |
| **资金面** | 20% | 换手率、量比、振幅、涨跌停状态、资金净流入天数、成交额 |
| **情绪面** | 15% | 近期涨跌天数、连涨/连跌趋势、价格相对位置、波动率、当日涨跌幅 |

### 预测输出
- **涨跌方向**：看涨 📈 / 看跌 📉 / 震荡 ➡️
- **预计涨跌幅**：百分比范围
- **置信度**：0-100%
- **四维评分**：各维度独立打分(0-100)
- **详细指标**：展开所有原始数据和技术指标

## 使用方法

### 方式一：直接在浏览器中使用
1. 将 `stock-predictor` 文件夹传到手机
2. 用浏览器打开 `index.html`
3. 输入股票名称或代码（如"贵州茅台"、"600519"）
4. 点击"预测"按钮

### 方式二：安装为PWA（推荐）
1. 在Chrome/Safari中打开应用
2. 点击浏览器的"添加到主屏幕"
3. 即可像原生App一样使用，支持离线缓存

### 方式三：部署到服务器
```bash
# 使用任意静态网站托管服务
# 推荐: GitHub Pages / Vercel / Netlify / Cloudflare Pages
```
